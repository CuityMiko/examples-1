# examples


plover示例应用，演示plover的核心功能。  

如果没有特别说明，都可以使用以下方式运行。

```shell
git clone git@github.com:ploverjs/examples.git
cd hello      # 切换到示例目录
npm install
npm start
```

访问 `http//127.0.0.1:4000`
